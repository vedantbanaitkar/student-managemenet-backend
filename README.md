# 📚 Student Management Backend

A minimal Node.js + Express.js backend for managing students, courses, and enrollments. Built for simplicity, with a modern frontend integration in mind.

---

## 🚀 Features

- ✅ Add and view students
- ✅ Add and view courses
- ✅ Enroll students in courses
- ✅ View enrolled courses per student
- ✅ API usage info endpoint

---

## 📦 Tech Stack

- Node.js
- Express.js
- In-memory data storage (for now)
- RESTful APIs
